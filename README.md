# Personal Portfolio Website

A single-page personal portfolio website built with HTML, CSS, and JavaScript.

This project is designed to:
- Present my background and skills
- Serve as my first public GitHub project
- Be deployable on any hosting or GitHub Pages

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Notes
- Fully static
- No backend
- Optimized for performance and readability

## GitHub Upload Steps
1. Create a new public repository
2. Name it (e.g. `portfolio-site`)
3. Upload all files (index.html, assets folder, etc.)
4. Commit with message: `Initial portfolio build`
